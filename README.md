backend-server-akka
===================

Lightweight Application Server for Akka actors, for Backend systems.


Distribution
============

backend-server-akka-<version>.zip


Dependencies
============

* Java SE 7 SDK (JDK)
* Akka 2.2.x and related dependencies
* Scala-2.11.x


Installation
============

Unzip the binary distribution file.
